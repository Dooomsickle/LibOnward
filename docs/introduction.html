Introduction

Onward, while being very popular and an online multiplayer game, has a surprising lack of security measures regarding modding the game. There's almost no obfuscation, nothing to prevent you from injecting anything, etc.

Codestage (the anticheat that Onward uses) even has its entire documentation online. As you could imagine, this is incredibly exploitable.

These guides will teach you some basics of Unity, the C# language itself, and a lot of Onward's core functionality. They expect you to be using MelonLoader (BepInEx is not covered in this).

Let's begin with learning how IL2CPP and the managed runtime are linked.
