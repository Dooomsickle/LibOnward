Networking

Onward uses its own networking system made by Downpour Interactive that appears to be built on Photon's PUN.

It's quite a complicated thing, so let's start simple.

DPIViews (OnwardPhotonViews)

Every networked GameObject in Onward has a component called an OnwardPhotonView. It's pretty much a proxy class for the DPIView component, but that's not important.
